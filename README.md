# TalaSound

	Ce dépôt contient les documents relatifs au projet informatique Tala Sound mené par Philomène Blot, Elise Costa et Laura Garda (élève de première année aux Mines de Paris) lors du second semestre, en 2020. Ce projet était encadré par Pierre Jouvelot et Luc Perera.

Ce dépôt contient :

- Rendu.ipynb : un Jupyter notebook avec le code, des commentaires et la démarche expliquée en détails.
-  : la documentation client
- des images, des fichiers audios, des vidéos (qui nourrissent notamment le notebook)
- ce README
